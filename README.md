**<center> Segmentation Study </center>**

>A study comparing segmentation models on different crops, genotypes and vegetative development

# Objectives

The main objective of this study is to compare the different segmentation models' outputs on corn ears, sorghum panicles and wheat spikes field images. As minor objective we have:

* Compare the performance different models across the different data collection dates, and genotypes;
* Develop a segmentation model that presents high metrics and does not present differences across genotypes and collection dates.
